# Estrutura de Projetos - LiveMode

## Princípios Básicos

Por enquanto, o básico é que os projetos mantenham um padrão de organização e tenham camadas separadas para cada responsabilidade.

## Organização por Responsabilidade

Cada projeto deve organizar seu código em camadas claras, separando responsabilidades:

- **Apresentação/UI**: Componentes, páginas, interfaces de usuário
- **Lógica de Negócio**: Regras de negócio, serviços, casos de uso
- **Dados**: Acesso a dados, repositórios, modelos
- **Infraestrutura**: Configurações, utilitários, integrações externas

## Padrão de Organização

Mantenha uma estrutura consistente dentro do mesmo projeto. A organização específica pode variar entre projetos, mas deve ser clara e seguir um padrão definido.
